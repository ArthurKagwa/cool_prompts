# 🪄 Vibe Coding Workshop: The Easter Egg Prompts

Welcome to the Vibe Coding workshop! Copy and paste the prompts below into your AI IDE to magically generate interactive, highly-styled web components and hidden secrets.

---

## 🟢 Prompt 1: The "Prompt-to-Reality" UI

**Goal:** Build a central command bar that changes the entire website's theme based on what you type.

**Frameworks/Tech:** Tailwind CSS, Matter.js

**Copy/Paste this to the AI:**

> "Create a dark-themed, minimalist web page using Tailwind CSS for all styling. In the exact center, put a large, glowing, modern text input bar that says 'Tell me the vibe...'. Write JavaScript that listens to what the user types in real-time. If they type 'cyberpunk', instantly change the page's Tailwind classes/CSS variables to a neon pink and cyan color scheme with a hacker-style monospace font. If they type 'boring', strip away all CSS styling so it looks like a default 1995 HTML page. If they type 'antigravity', use the Matter.js physics engine library to make the input bar detach and float around the screen using basic physics and boundary collisions."

## 🟢 Prompt 2: The Ghost in the Machine (Auto-Typing)

**Goal:** Create text that looks like an invisible AI is typing it out live, with a funny interruption mechanic.

**Frameworks/Tech:** Vanilla JavaScript (or Typed.js)

**Copy/Paste this to the AI:**

> "Add a paragraph of text below the main input bar that explains what 'Vibe Coding' is. However, do not display the text immediately. Use vanilla JavaScript (or a library like Typed.js if you prefer) to 'type' this text out one character at a time, like a ghost is typing it at 800 words per minute. Make it look realistic by occasionally pausing or 'backspacing' a fake typo. Add an Easter egg: if the user clicks anywhere on the typing text, immediately stop the animation, print the exact phrase '$$USER INTERRUPT - RECALIBRATING VIBES...$$', and change the font of that specific paragraph to Comic Sans."

## 🟢 Prompt 3: The Lo-Fi "Focus Mode" Button

**Goal:** Add a hidden ambient mode for deep-focus coding.

**Frameworks/Tech:** Tailwind CSS, Lucide Icons, HTML5 Audio

**Copy/Paste this to the AI:**

> "Import Lucide Icons (or FontAwesome) and add a tiny, subtle, minimalist coffee cup icon fixed to the bottom-right corner of the screen with an opacity of 0.3. When the user hovers over it, make it glow. When the user clicks it, trigger 'Focus Mode': use Tailwind CSS to smoothly transition the website's background into a slow-moving, animated, soothing gradient (like deep purples and blues). Also, embed a hidden HTML5 \`<audio>\` player that loops a chill lo-fi hip hop beat (use a placeholder audio URL for now). Clicking the cup again should turn everything off."

## 🟢 Prompt 4: The Konami Code "Matrix" Takeover

**Goal:** Build a classic video game cheat code that triggers a visual hacker takeover.

**Frameworks/Tech:** Vanilla HTML5 \`<canvas>\`, Vanilla JavaScript

**Copy/Paste this to the AI:**

> "Implement a global keyboard event listener in vanilla JavaScript that tracks the user's keystrokes and listens for the classic Konami Code (ArrowUp, ArrowUp, ArrowDown, ArrowDown, ArrowLeft, ArrowRight, ArrowLeft, ArrowRight, b, a). When this exact sequence is successfully typed, trigger a dramatic Easter egg: make the screen go completely black, and use a vanilla HTML5 \`<canvas>\` element to create a 'Matrix Digital Rain' falling effect over the entire screen. However, instead of random characters falling, make the falling text rapid-fire AI coding prompts like \`generate_component()\`, \`fix_my_bugs()\`, and \`optimize_loop()\`."

## 🟢 Prompt 5: Console Log Secrets

**Goal:** Reward curious developers who peek under the hood in the browser's developer tools.

**Frameworks/Tech:** Vanilla JavaScript, CSS (in-console)

**Copy/Paste this to the AI:**

> "Write a vanilla JavaScript self-executing function that fires as soon as the page loads. It should use \`console.log()\`, but I want you to apply CSS styling directly to the console output using the \`%c\` formatter. Print out a massive, colorful ASCII art logo of an alien or a robot. Below the art, print a styled, glowing message that says: 'Ah, a fellow developer! You found the secret console. Type the passcode \`sudo vibe\` into the main page input bar to unlock the developer mode.' (And then, make sure to add logic to the main input bar to listen for \`sudo vibe\` and show a secret success message!)."
